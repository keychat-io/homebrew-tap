# Homebrew Tap for Keychat

Homebrew formulae for [Keychat](https://github.com/keychat-io/keychat-protocol) tools.

## Install

```bash
brew tap keychat-io/tap
brew install keychat
```

## Upgrade

```bash
brew upgrade keychat
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `keychat` | Encrypted messaging CLI over Nostr with Signal protocol |
